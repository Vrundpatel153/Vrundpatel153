## âœ¨ Eye-Candy â€” Animated Contribution "Feel"

GitHub does not allow animating the official contribution graph â€” but we *simulate* an animated effect with an SVG that updates frequently.

### Snake animation (copy & paste)

1. Add a GitHub Action named `.github/workflows/snake.yml`. Use `Platane/snk` (or similar) to generate an SVG that mimics the heatmap and adds a moving "snake".

**Embed** the generated SVG in your README like this (replace `YOUR_USERNAME`):

```md
![snake animation](https://raw.githubusercontent.com/YOUR_USERNAME/YOUR_USERNAME/output/github-contribution-grid-snake.svg)
```

2. Place the `snake.yml` into your repo's `.github/workflows/` and the action will write the SVG to `output/` automatically.

**Example `snake.yml` (minimal):**

```yaml
name: GitHub Contribution Snake
on:
  schedule:
    - cron: '0 0 * * *'  # daily
  push:
    branches: [ main ]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: Generate snake
        uses: Platane/snk@main
        with:
          username: YOUR_USERNAME
          output: output/github-contribution-grid-snake.svg
      - name: Commit SVG
        uses: stefanzweifel/git-auto-commit-action@v4
        with:
          commit_message: "chore: update contribution snake"
          file_pattern: output/github-contribution-grid-snake.svg
```

> The result is a living SVG â€” crisp, small, and customizable. The snake path gives the impression of animation.
